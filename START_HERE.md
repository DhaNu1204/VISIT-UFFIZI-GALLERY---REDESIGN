## ⚡ QUICK START (2 Options)

### OPTION 1: Git Deployment (Recommended - 10 minutes)
**Best for:** Ongoing updates and automation

1. Log in to **Hostinger hPanel**
2. Go to **Git Version Control**
3. Connect this GitHub repo:
   ```
   https://github.com/DhaNu1204/VISIT-UFFIZI-GALLERY---REDESIGN.git
   ```
4. Set branch: `main`
5. Build command: `npm install && npm run build`
6. Output directory: `out`
7. Click **Pull** and deploy!

**✅ Full instructions in `HOSTINGER_DEPLOYMENT.md`**

---

### OPTION 2: Manual Upload (Simpler - 15 minutes)
**Best for:** One-time deployment

1. Log in to **Hostinger hPanel**
2. Go to **File Manager**
3. Navigate to `/public_html`
4. **BACKUP** existing WordPress files
5. **DELETE** old files
6. **UPLOAD** everything from the `out` folder
7. **UPLOAD** the `.htaccess` file from project root

**✅ Full instructions in `HOSTINGER_DEPLOYMENT.md`**
